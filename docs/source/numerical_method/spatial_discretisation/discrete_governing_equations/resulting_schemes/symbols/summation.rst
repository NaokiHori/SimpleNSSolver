#########
Summation
#########

At :math:`\gcs{1}` cell faces:

.. math::

    \sumxf q
    \equiv
    \vat{q}{\frac{1}{2}}
    +
    \vat{q}{\frac{3}{2}}
    +
    \cdots
    +
    \vat{q}{\ngp{1} - \frac{1}{2}}
    +
    \vat{q}{\ngp{1} + \frac{1}{2}}.

At :math:`\gcs{1}` cell centers:

.. math::

    \sumxc q
    \equiv
    \vat{q}{1}
    +
    \vat{q}{2}
    +
    \cdots
    +
    \vat{q}{\ngp{1} - 1}
    +
    \vat{q}{\ngp{1}}.

At :math:`\gcs{2}` cell faces:

    +
    \vat{q}{\ngp{2} - \frac{1}{2}}.
